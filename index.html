Atone

En cours de réalisation

Jeu vidéo

Contexte : Jeu réalisé en cinq mois avec une équipe de quatorze personnes dans le cadre du Master. Collaboration entre première année, deuxième année ainsi que des élèves d’un diplôme d’établissement en sound design et composition.

Description : Les Minors sont une race extraterrestre colonisée par les humains. Vous incarnez l'un d'entre eux et vous subirez l’épreuve du Diapason ayant pour objectif de vous replonger dans une transe aliénante servant à garder le contrôle de la population.

Gameplay : Ce jeu de rythme à la première personne vous demandera de surmonter les différents obstacles se dressant devant vous en effectuant toutes vos actions suivant un rythme défini.

Gestion de projet : Co-gestion de tout le projet à deux, organisation des réunions, mis à jour de l’avancement du projet et de chacun, relation avec les encadrants de l’université.

Game Design : Co-conception du game design, écriture de documents.

Level Designer : Co-conception des niveaux du jeu.

Outils : Unity comme moteur de jeu, Monday comme gestion de projet, Photoshop pour faire des croquis.

Fresque du climat

Serious Game

Contexte : Prototype d'un serious game réalisé à partir d’un premier exemplaire existant lors d’un stage de cinq mois dans le studio Ikigai. Je travaillais seul sur le projet avec la supervision et les conseils d’un directeur créatif et d’un lead programmeur.

Description : Le projet est une adaptation de La Fresque du Climat, un atelier de sensibilisation écologique, en application mobile. Vous devrez fusionner des éléments se trouvant dans la fresque pour débloquer les cartes de l’atelier original.

Gameplay : Vous devrez fusionner des éléments grâce à un système de drag & drop. Vous découvrirez ainsi d’autres éléments qui permettront de révéler des cartes. Le but est de posséder toutes les cartes.

Game Design : Remaniement du design original

Programmeur : Programmation du second prototype à partir d’une base existante.

Outils : Unity comme moteur de jeu, Visual Studio comme outil de programmation. Google Sheet pour le game design.

Edmond and The Fraudulent Ghost

Jeu vidéo

Contexte : Jeu réalisé lors de la Global Game Jam 2022 avec une équipe de huit, en travaillant partiellement durant le week-end.

Description : Vous incarnez un huissier venu pour une saisie immobilière dans un manoir hanté. Il faudra échapper au fantôme qui ne réagit qu’à la lumière et au son.

Gameplay : Dans ce jeu, vous devez atteindre la sortie du manoir sans vous faire attraper par le fantôme. Ce dernier se rapproche si vous percutez des objets en faisant du bruit et si vous allumez la lumière. Il faudra donc éclairer avec parcimonie pour se rappeler du chemin sans attirer le fantôme.

Game Design : Co-conception du game design.

Level Designer : Conception papier et intégration des niveaux.

Outils : Unity pour le moteur de jeu, Photoshop pour dessiner les niveaux.

Description : Vous incarnez un humain enlevé par un génie du mal, concepteur de robots maléfiques. Il faudra vous enfuir en évitant les robots. Vous pourrez dialoguer avec l’antagoniste sur un ton décalé et humoristique.

La Nouvelle

Jeu vidéo

Contexte : Projet principal du Master. Réalisé à neuf, en cinq mois, mélangeant des étudiants de premières et deuxièmes années. L'idée initiale a été conçu par les deuxièmes années qui avaient déjà imaginé le concept et ses principales mécaniques.

Description : Vous incarnez Enola, une collégienne subissant du harcèlement scolaire. Vous affronterez le regard des autres grâce à Lucie, une autre collégienne cohabitant dans votre tête grâce au pouvoir d’une mystérieuse barrette.

Gameplay : Le jeu se décompose en deux phases. La phase narrative où vous pouvez dialoguer avec des élèves et la phase de plateforme où vous devrez aller au bout du niveau en transformant les insultes des harceleurs en plateforme.

Game Design : Co-conception des fonctionnalités principales de la phase de plateforme, ainsi que de documents de game design.

Animateur 2D : Animation de tous les personnages du jeu.

Artiste VFX : Fumée noire représentant le vide.

Level Builder : Assemblage des éléments graphiques du niveau plateforme pour représenter le fait que l'angoisse d'Enola s'intensifie.

Outils : Unity pour réaliser le level design, les animations 2D et le VFX.

Jeu vidéo

Description : Vous incarnez un trappeur qui vit seul dans la taïga, suite au décès de sa femme. Vous allez devoir l’accompagner à travers chaque étape du deuil.

Game Design : Co-conception du game design à sept. Création des patterns de respirations.

Graphismes : Réalisation de certains VFX : Intempéries (Neige, Blizzard), Eau, Post-process, Skybox, Pattes de loups...

Outils : Unity comme moteur de jeu, Visual Studio comme IDE (Integrated Development Environment). Git comme logiciel de gestion de version. Photoshop pour certains effets.

Coup de Grisou !

Jeu vidéo

Contexte : Réalisé lors de l’Unijam 2019, une jam réalisée en collaboration avec une école de programmeur. Nous étions sept, avec comme thème «Ne craignez pas d’être lent, craignez seulement d’être à l’arrêt».

Description : Vous contrôlez Stakanov, un mineur qui doit extraire le plus de bloc de cristaux d’une mine, avant la fin du temps imparti. Cependant, les cristaux relâchent du gaz en permanence et plus encore quand vous les minez.

Gameplay : Il fout donc bouger constamment pour ne pas être exposé au gaz tout en minant les cristaux. Attention, si vous les minez trop vite, ils relâcheront trop de gaz et vous vous asphyxierez.

Game Design : Équilibrage du jeu.

Level Design : Co-conception du level design et grayboxing.

Outils : Unity comme moteur de jeu. MagicaVoxel pour build le level.

Décroissance

Jeu de société

Contexte : Jeu de société réalisé en une semaine, à quatre, dans le cadre de ma Licence. Notre thème était le "réchauffement climatique".

Description : Le jeu se situe en l’an 2100, l’effondrement de nos sociétés a eu lieu et vous êtes les derniers survivants. Vous avez survécu avec l’aide des anciennes machines qui sont efficaces mais polluantes. Leurs pollutions engendrent des catastrophes naturelles qui finiront par vous tuer. Vous allez donc devoir trouver une solution pour survivre durablement.

Gameplay : Vous agissez en coopération, en équipe de quatre. Vous allez devoir récoltez des ressources sur une carte, afin de construire des innovations résilientes, pour remplacer les anciennes machines polluantes. Le but est d’atteindre un minimum de pollution par tour, la difficulté étant que les innovations sont moins rentables et restreignent vos moyens pour dépolluer et survivre.

Game Design : Co-conception du game design à quatre.

Graphisme : Réalisation des cartes catastrophes et autres détails sur Photoshop.

Outils : Photoshop pour les graphismes.

Jeu de société éducatif

Contexte : Jeu réalisé en Service Civique, à quatre, dans le but de sensibiliser à l’environnement.

Description : Le jeu se présente comme un Uno avec des éco-gestes sur les cartes

Gameplay : Règles classiques du Uno. Cependant, après un certain nombre de tours, chacun doit regarder ses cartes. S’il ne fait pas, dans la réalité, l'éco-geste indiqué sur la carte, alors on ajoute, sur une feuille, les points marqués sur la carte. A la fin, celui qui a le plus de point a perdu.

Game Design : Co-conception du game design.

Graphismes : Réalisation des cartes sur Photoshop.

Polish a Knife

Serious Game

Contexte : Serious Game, réalisé à quatre, lors de mon IUT, dans le cadre d’un projet tutoré puis d’un stage de fin d'études.

Description : Simulation de polissage d’un couteau en réalité virtuelle, le but étant d’accroître la sécurité des apprentis et de réduire le coût en ressource nécessaire à la formation.

Gameplay : Vous êtes assis sur un tabouret, en face de la machine de polissage. Vous devez prendre le couteau, qui sera représenté par votre manette. Il faudra ensuite mettre les bons outils, dans le bon ordre, sur la machine, puis présenter le couteau de la bonne manière de façon à reproduire le polissage.

Game Design : Co-conception du game design à quatre.

Programmation : Co-réalisation de la programmation.

Graphismes : Participation à la modélisation 3D de la machine, de l’établi, etc. Création de shaders pour faire tourner les textures.

Gestion de Projet : TODO List et respect des dead lines.

Outils : Unity et Visual Studio comme moteur de jeu et IDE. 3DS Max pour la modélisation. Photoshop pour les textures et autres. HTC Vive comme casque VR.

Escape Room

Jeu vidéo

Contexte : Escape Game, réalisé à quatre, en dix semaines, dans le cadre de mon IUT.

Description : Vous êtes enfermés dans un manoir et vous devez résoudre des énigmes pour vous échapper avant la fin du temps imparti.

Gameplay : La caméra est en First Person. Il faut utiliser les bons objets dans l’inventaire, aux bons endroits, et résoudre les énigmes pour s’échapper. Il y a trois salles différentes, comportant chacune des énigmes.

Game Design : Conception des énigmes et co-conception du reste.

Programmation : Co-réalisation de la programmation à deux.

Sound Design : Enregistrement des sons à deux, puis implémentation.

Outils : Unity et Visual Studio comme moteur de jeu et IDE.
